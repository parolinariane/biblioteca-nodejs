# Sistema da Biblioteca Universitária

Projeto desenvolvido em JavaScript + Node.js para praticar lógica de programação, estruturas condicionais, laços de repetição e interação via terminal.

---

## Funcionalidades

- Registro de devolução de livros
- Cálculo automático de multa
- Processamento de múltiplos livros
- Menu interativo no terminal
- Entrada de dados pelo usuário
- Terminal estilizado com cores usando Chalk

---

## Tecnologias utilizadas

- JavaScript
- Node.js
- Chalk

---

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/parolinariane/biblioteca-nodejs.git
```

Acesse a pasta:

```bash
cd biblioteca-nodejs
```

Instale as dependências:

```bash
npm install
```

Execute o sistema:

```bash
node index.js
```
